<h1>Simulate Steam game sessions and increase playtime via idle mode.</h1>


<h2>⚠️ WARNING</h2>
This tool is for <b>educational purposes only</b>.<br>
Use at your <b>own</b> risk.

<h2>🚀 INSTALL</h2>
<ul>
  <li>First you need install Node.js appropriate for your system (if you don't have it installed) by this link https://nodejs.org/en/download</li>
</ul>

```bash 
git clone https://github.com/vLucqy/steam-playtime-booster.git 
```

```bash 
cd steam-playtime-booster 
```

```bashe
npm install
```

<h2>⚡ RUN</h2>

```bash
node index.js
```


<h2>🎮 USAGE FLOW</h2>
<ol>
  <li>First, open index.js. Write your username and password inside the quotation marks on lines 13 and 14, then save the file

  ```js
      accountName: "YOUR_STEAM_USERNAME",
      password: "YOUR_STEAM_PASSWORD",
  ```
  </li>
  <li>Before running the project:
    <ul>
      <li>Open the Steam Mobile App (if Steam Guard is enabled)</li>
      <li>Approve the login request</li>
      <li>Then enter the Steam Guard code</li>
    </ul>
  </li>
  <li>Run the file using the RUN instructions section</li>
  <li>To find a game App ID:
    <ul>
      <li>Go to the Steam Store page of the game</li>
      <li>In the URL, after /app/, copy the number
        <br>
        <img src="https://github.com/vLucqy/steam-playtime-booster/blob/main/guide/appId.png?raw=true" width="500px">
      </li>
    </ul>
  </li>
  <li>Now the project will run easily and start farming playtime</li>
</ol>
